fourierToSeasonal[source]

fourierToSeasonal(mod)

{% endraw %}

This function transforms the seasonal component of a model from fourier form into standard seasonal components. For example, if seasPeriod = [7], then this would return a vector of length $7$, with each of the seasonal effects. The seasonal effects represent a deviation from the mean, so they will sum to $0$.
